What Is Github ?

- GitHub is a web-based platform for version control and collaboration.
- It is built around the Git system.
- Developers use GitHub to store, manage, and track changes to their code.
- It enables multiple people to work together on projects.
- Projects can be shared publicly or kept private.
- GitHub offers tools for issue tracking, code review, and project management.
- It is widely used for both open-source and private software development.
